# 💰 VidIQ — Monetization Strategy

## Revenue Model: Freemium + API Access

### Free Tier (Core)

All users get:
- 3 video analyses per day
- Standard scene timeline output
- Basic sharing (copy text / download)
- Analysis history (last 7 days)
- Max 30-second videos

### Premium Tier — "Unlimited Vision" ($4.99/month or $39.99/year)

Premium users get:
- **Unlimited analyses**
- **Videos up to 10 minutes**
- **Custom prompts** (guide what AI looks for)
- **Multi-model comparison** (run same video through multiple models)
- **Export as PDF / SRT / JSON**
- **Extended history** (unlimited retention)
- **Priority processing** (faster queue)
- **Ad-free experience**

## API Access (Phase 2)

### Developer Tier ($29/month)
- 100 API calls/month
- All models available
- Webhook notifications
- JSON/structured output
- Rate: $0.29/analysis

### Business Tier ($99/month)
- 500 API calls/month
- Priority processing
- SLA guarantees
- Dedicated support

### Enterprise (Custom)

## Key Pricing Decisions

1. **Free tier must feel useful** — 3 analyses/day is enough to demonstrate value
2. **Premium is about power & scale** — longer videos, custom prompts, batch
3. **API revenue is the real business** — B2B, recurring, scalable
4. **Never paywall basic analysis** — that kills adoption
